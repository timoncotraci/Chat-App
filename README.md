# 💬 Chat App

A simple real-time chat application built with **Node.js**, **WebSocket (ws)**, **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features
- Real-time messaging using WebSocket  
- Simple and clean UI  
- Can be accessed via phone (on same Wi-Fi network)  
- Runs locally or can be hosted online  

## 🛠️ Setup Instructions
```bash
git clone https://github.com/timoncotraci/Chat-App.git
cd Chat-App
npm install
node server.js
